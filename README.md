### 프로젝트
Nomadcoders Vanilla-JS Challenge

### 프로젝트 설명
JavaScript의 기본 문법(Variables, Functions, Arrays, Objects, Conditionals 등)을 습득하여,
Chrome 확장프로그램 Momentum을 클론코딩 합니다.

### 개인 목표
JavaScript 기본 문법 습득

### 사용한 언어
JavaScript

### 구현한 기능
- 실시간 시계
- 로컬스토리지를 사용한 로그인
- 로컬 스토리지를 사용한 투두리스트
- 랜덤 배경 이미지
- 날씨와 위치정보 얻기 (Open API 사용)
- GitHub 페이지에 배포
